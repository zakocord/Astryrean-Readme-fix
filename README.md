## Features
-   Discord Token Info
    -   Nitro
    -   Badges
    -   Email
    -   Phone
    -    
-   System Info
    -   Host, User
    -   Disk Info
    -   CPU, GPU, RAM, Hwid
    -   Screenshot
    -
